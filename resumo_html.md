# Resumo sobre HTML

## O que é HTML?
O HTMl é uma linguagem que pode ser usado em uma programação, porém não é uma programação por sí própria, pois ela não possui funcionalidades como função, loop dentre outros. 

## Principais Tags
"p" é usada em parágrafos
"img" adiciona imagens
"a" para por atributos que transformam elementos em clicáveis.
<p> E outras tags que são responsáveis pela estruturação de um site como:</p>
"head"
"body"
"meta"
"div"
